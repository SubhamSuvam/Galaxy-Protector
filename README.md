# Galaxy-Protector

A fast-paced 2D space shooter game built using pure HTML, CSS, and JavaScript.
Fight waves of enemy spaceships, collect Galaxy Coins (GC), upgrade your abilities, and survive as long as possible.
Developed by Subham.

# Features
🚀 Smooth 2D space shooter gameplay
🛡️ Shield power-up system
❤️ Upgradeable health/life system
💰 Galaxy Coin (GC) economy
🛒 In-game shop with upgrades
📊 Player statistics tracking
🔊 Background music and sound effects
📱 Mobile touch joystick controls
💾 Save system using Local Storage
🌌 Animated loading screen
🎨 Responsive futuristic UI
👾 Progressive enemy spawning system

# Gameplay
Destroy enemy ships to earn kills and Galaxy Coins.
Every 2 kills rewards you with GC.
Use GC in the shop to:
Buy shields
Upgrade shield duration
Increase maximum lives
Activate shields during gameplay to avoid damage.
Survive as long as possible and beat your high score.

# Save System
The game automatically saves:
Player stats
GC balance
Purchased upgrades
Audio settings
Shield inventory
All data is stored locally in the browser using LocalStorage.
